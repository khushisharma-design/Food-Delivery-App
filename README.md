# BrewHouse — Demo Coffee Shop

More realistic single-page Coffee Shop demo built with HTML, CSS and JavaScript. The updated app includes detailed menu items, quantity controls, service fee calculation, and a café-style checkout flow. Prices are shown in AED.

Open the app locally at:

- [BrewHouse/index.html](index.html#L1)

To serve via a local HTTP server (recommended):

```powershell
Set-Location -LiteralPath 'C:\Users\hp\Food Delivery App'
python -m http.server 8000
# then open http://localhost:8000/index.html
```
